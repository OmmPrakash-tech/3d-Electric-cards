# 🎴 3D Electric Cards

A visually engaging **3D Electric Card UI** built using pure **HTML** and **CSS**. This project showcases modern design techniques like 3D transformations, hover effects, and glowing electric-style animations to create interactive cards.

---

## 🚀 Features

* ⚡ Smooth 3D hover effects
* 🎨 Electric glow animation
* 📱 Responsive design
* 💡 Clean and minimal UI
* 🧩 Built with only HTML & CSS (no JavaScript)

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**

  * Flexbox
  * Transforms (3D)
  * Transitions & Animations

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/OmmPrakash-tech/3d-Electric-cards.git
   ```

2. Navigate to the project folder:

   ```bash
   cd 3d-Electric-cards
   ```

3. Open `index.html` in your browser.
